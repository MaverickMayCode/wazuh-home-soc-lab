# wazuh-home-soc-lab
Home SOC Lab using Wazuh SIEM to detect and investigate simulated security events.

## Objective
Build a home Security Operations Center (SOC) lab using Wazuh to ingest logs, detect threats, and investigate simulated cyber attacks.

## Why I Built This
I recently earned my CompTIA Security+ and am working toward a SOC Analyst role.
This project is meant to demonstrate hands-on experience with SIEM deployment, log analysis, and incident response.

## Lab Goals
- Deploy Wazuh SIEM
- Ingest logs from Linux endpoints
- Simulate real-world attacks using Kali Linux
- Investigate alerts and write incident reports

## Tools Used
- Wazuh (SIEM/XDR)
- Ubuntu Server
- Kali Linux
- VirtualBox

## Project Status
- In Progress – Initial lab setup complete. Three simulations run.
