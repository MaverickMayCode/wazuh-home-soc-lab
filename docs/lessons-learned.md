# Lessons Learned

## Technical Lessons

- Configuring VirtualBox networking correctly (Bridged vs NAT)
- SIEM requires a log ingestion to function - most times from an agent
- Agents are essential for collecting endpoint data and generating logs

## Troubleshooting Lessons

- VM crashes due to virtual memory misallocation can be resolved by restarting PC and discarding saved state
- Catching file name mismatches that cause execution errors in Linux

## Security Insights

- Multiple failed login attempts are a key indicator of brute-force attacks
- SIEM tools rely on pattern recognition, not just single events
- Privilege escalation is typically flagged as a high-risk/high-score activity
